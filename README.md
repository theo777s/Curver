A free, lightweight tool for creating super smooth corner paths for use in your designs.

Generate superellipse, clothoid, squircle, and arc corner curves — export as SVG for use in Figma, Affinity, Illustrator, Photoshop, and Inkscape.

## Enjoy here:

[https://theo777s.github.io/Curver](https://theo777s.github.io/Curver)
